# Questions on Lists

## Question 1

Concatenate two lists index-wise.

Example:

```txt
list1 = ["M", "ho", "to", "i", "war"]
list2 = ["y", "me", "wn", "s", "angal"]
```

Expected Output:

```txt
['My', 'hometown', 'is', 'warangal']
```

## Question 2

Concatenate two lists in the following order.

Example:

```txt
list1 = ["Hi ", "Hey "]
list2 = ["Madam", "Sir"]
```

Expected Output:

```txt
['Hi Madam', 'Hi Sir', 'Hey Madam', 'Hi Sir']
```
